# Manual de la Escuela de la PLAPA

Este repositorio contiene el código fuente en LaTeX y el PDF oficial del manual desarrollado para la Pastoral Latinoamericana de Acompañamiento y Prevención de las Adicciones (PLAPA) del Consejo Episcopal Latinoamericano y Caribeño (CELAM)

## Propósito
Este material ha sido desarrollado como el marco pedagógico y técnico para la formación en el acompañamiento y prevención de adicciones en las comunidades de la región, bajo las orientaciones de la Pastoral Latinoamericana de Acompañamiento y Prevención de las Adicciones (PLAPA).
